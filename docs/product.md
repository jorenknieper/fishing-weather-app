# Product: Fishing Weather App

## Problem

Generic weather apps are not optimized for anglers.

Anglers care about:

- Pressure
- Pressure trends
- Temperature
- Humidity
- Rain
- Wind
- Moon phase
- Sunrise and sunset
- Local conditions near the fishing spot

## MVP Goal

Build a static dashboard that shows current fishing-relevant weather for Roeselare.

## MVP Features

- Show current temperature
- Show apparent temperature
- Show humidity
- Show sea-level pressure
- Show surface pressure
- Show rain
- Show wind speed
- Show wind direction
- Show last update timestamp

## Out of Scope

- User accounts
- Database
- AWS
- IoT devices
- Native iOS or Android apps
- Advanced fish activity scoring
- Historical charts
